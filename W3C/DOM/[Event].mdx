## Event Phases

The event goes through the following phases:

1. **Capture Phase**: Event travels from the window down to the target
2. **Target Phase**: Event reaches the target element
3. **Bubble Phase**: {event.bubbles ? 'Event bubbles up from target to window' : 'This event does not bubble'}
