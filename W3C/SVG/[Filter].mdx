## Input Sources

The `in` and `in2` attributes can accept the following input sources:

- **SourceGraphic**: The original element that the filter is applied to
- **SourceAlpha**: The alpha channel of the source graphic
- **BackgroundImage**: The background image at the filter region
- **BackgroundAlpha**: The alpha channel of the background image
- **FillPaint**: The fill paint of the element
- **StrokePaint**: The stroke paint of the element
- Previous filter primitive results (referenced by their `result` attribute)
