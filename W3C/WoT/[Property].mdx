## Usage Examples

### Read Property

```javascript
// HTTP GET example
const response = await fetch('https://thing.example.com/properties/{property.name}');
const value = await response.json();
console.log('Current value:', value);
```

```javascript
// Using WoT Scripting API
const thing = await WoT.consume(thingDescription);
const value = await thing.readProperty('{property.name}');
console.log('Property value:', value);
```

### Write Property

```javascript
// HTTP PUT example
const newValue = {property.default};
const response = await fetch('https://thing.example.com/properties/{property.name}', {
  method: 'PUT',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(newValue)
});
```

```javascript
// Using WoT Scripting API
await thing.writeProperty('{property.name}', newValue);
```

### Observe Property

```javascript
// Using WoT Scripting API with observable property
thing.observeProperty('{property.name}', async (value) => {
  console.log('Property changed:', value);
});
```

```javascript
// Using Server-Sent Events (SSE)
const eventSource = new EventSource('https://thing.example.com/properties/{property.name}/observe');
eventSource.onmessage = (event) => {
  console.log('Property update:', JSON.parse(event.data));
};
```
