### Invoke Action (Basic)

```javascript
// Using WoT Scripting API
const thing = await WoT.consume(thingDescription);
const result = await thing.invokeAction('{action.name}');
```

## Asynchronous Action Handling

```javascript
// For long-running actions
const actionHandle = await thing.invokeAction('{action.name}', inputData);

// Poll for status (if supported)
const status = await actionHandle.getStatus();
console.log('Action status:', status);

// Wait for completion
const result = await actionHandle.wait();
console.log('Final result:', result);
```

## Error Handling

```javascript
try {
  const result = await thing.invokeAction('{action.name}', inputData);
  console.log('Success:', result);
} catch (error) {
  console.error('Action failed:', error.message);
  if (error.code === 'TIMEOUT') {
    console.log('Action timed out');
  } else if (error.code === 'INVALID_INPUT') {
    console.log('Invalid input parameters');
  }
}
```
