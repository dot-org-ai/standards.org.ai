### Advanced Usage

<CodeExample language="sparql" title="Advanced Example">
{`# Advanced query demonstrating ${keyword.name} with multiple features
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>

  ?item rdfs:label ?label .

  }
}
ORDER BY ?label
` : keyword.category === 'Modifier' ? `

WHERE {
  ?instance a ?type .
}
GROUP BY ?type
${keyword.name.toUpperCase()}
HAVING(?count > 5)
` : `
SELECT ?resource ?property
WHERE {
  ?resource ?property ?value .

### Browser Support

SPARQL queries are executed server-side by SPARQL endpoints. Client-side JavaScript libraries like SPARQL.js can be used to construct and execute queries from web applications.
