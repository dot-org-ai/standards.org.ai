## Inference and Reasoning

RDFS reasoning engines apply the following inferences for this property:

### Domain Inference

If `?x {Property.uri} ?y` and this property has domain `D`, then `?x rdf:type D`.

### Range Inference

If `?x {Property.uri} ?y` and this property has range `R`, then `?y rdf:type R`.

### Sub-Property Inheritance

If `P rdfs:subPropertyOf {Property.uri}` and `?x P ?y`, then `?x {Property.uri} ?y`.

## Semantic Constraints

Understanding the semantics of this property:

1. **Open World Assumption**: Absence of a triple doesn't mean it's false
2. **Multiple Values**: Unless otherwise stated, properties can have multiple values
3. **Type Inference**: Domain and range assertions enable automatic type inference
4. **Inheritance**: Sub-properties inherit domain and range constraints
