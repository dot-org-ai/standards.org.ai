<section className="dcterms-overview">
  <h2>About Dublin Core Terms</h2>
  <div className="info-box">
    <p>
      <strong>Dublin Core Terms</strong> (dcterms:) extends the original Dublin Core Element Set
      with additional properties and classes. It provides more specific terms and uses
      formal domains and ranges.
    </p>
    <h4>Key Differences from DC Elements</h4>
    <ul>
      <li><strong>Namespace:</strong> Uses http://purl.org/dc/terms/ instead of /elements/1.1/</li>
      <li><strong>Refinements:</strong> Many terms refine the original 15 elements</li>
      <li><strong>Typed Values:</strong> Specifies formal ranges (URIs vs. literals)</li>
      <li><strong>Classes:</strong> Includes class definitions for resource types</li>
    </ul>
    <h4>Common Term Categories</h4>
    <ul>
      <li><strong>Properties:</strong> Metadata elements (like dcterms:created, dcterms:modified)</li>
      <li><strong>Classes:</strong> Resource types (like dcterms:Dataset, dcterms:Agent)</li>
      <li><strong>Vocabulary Encoding Schemes:</strong> Controlled vocabularies</li>
      <li><strong>Syntax Encoding Schemes:</strong> Format specifications</li>
    </ul>
  </div>
</section>
