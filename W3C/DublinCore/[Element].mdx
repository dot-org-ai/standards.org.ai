<section className="dc-overview">
  <h2>About Dublin Core Elements</h2>
  <div className="info-box">
    <p>
      The <strong>Dublin Core Metadata Element Set</strong> is a vocabulary of fifteen properties
      for use in resource description. It provides a simple and standardized set of conventions
      for describing resources.
    </p>
    <h4>The 15 Core Elements</h4>
    <ul>
      <li><strong>dc:title</strong> - A name given to the resource</li>
      <li><strong>dc:creator</strong> - An entity primarily responsible for making the resource</li>
      <li><strong>dc:subject</strong> - The topic of the resource</li>
      <li><strong>dc:description</strong> - An account of the resource</li>
      <li><strong>dc:publisher</strong> - An entity responsible for making the resource available</li>
      <li><strong>dc:contributor</strong> - An entity responsible for making contributions</li>
      <li><strong>dc:date</strong> - A point or period of time associated with the resource</li>
      <li><strong>dc:type</strong> - The nature or genre of the resource</li>
      <li><strong>dc:format</strong> - The file format, physical medium, or dimensions</li>
      <li><strong>dc:identifier</strong> - An unambiguous reference to the resource</li>
      <li><strong>dc:source</strong> - A related resource from which it is derived</li>
      <li><strong>dc:language</strong> - A language of the resource</li>
      <li><strong>dc:relation</strong> - A related resource</li>
      <li><strong>dc:coverage</strong> - Spatial or temporal topic, applicability</li>
      <li><strong>dc:rights</strong> - Information about rights held in and over the resource</li>
    </ul>
  </div>
</section>
