## Valid Formats

{dataType.category === 'length' && `
### Absolute Lengths
- **px** - Pixels (1px = 1/96th of 1 inch)
- **cm** - Centimeters
- **mm** - Millimeters
- **in** - Inches (1in = 96px)
- **pt** - Points (1pt = 1/72nd of 1 inch)
- **pc** - Picas (1pc = 12pt)

### Relative Lengths
- **em** - Relative to font-size of the element
- **rem** - Relative to font-size of the root element
- **ex** - Relative to x-height of the font
- **ch** - Relative to width of "0" character
- **vw** - 1% of viewport width
- **vh** - 1% of viewport height
- **vmin** - 1% of viewport's smaller dimension
- **vmax** - 1% of viewport's larger dimension
- **%** - Percentage relative to parent
`}

{dataType.category === 'color' && `
### Named Colors

- Extended colors: rebeccapurple, cornflowerblue, etc.

- 3-digit: #RGB
- 6-digit: #RRGGBB
- 4-digit with alpha: #RGBA
- 8-digit with alpha: #RRGGBBAA

## Best Practices

- Use appropriate units for the context (relative vs absolute)
- Consider accessibility when using colors (contrast ratios)
- Prefer relative units for responsive designs
- Use consistent units throughout your stylesheet
- Validate values are within acceptable ranges

{dataType.category === 'length' && `
- Use relative units (rem, em) for typography
- Use viewport units (vw, vh) for responsive layouts
- Avoid mixing units unnecessarily
- Consider using CSS custom properties for reusable values
`}

{dataType.category === 'color' && `
- Ensure WCAG AA/AAA contrast ratios for text
- Use HSL for easier color manipulation
- Consider using CSS custom properties for color schemes
- Test colors for color-blind users
- Use transparent keyword instead of rgba(0,0,0,0)
`}

{dataType.category === 'time' && `

## Common Mistakes

- Using incorrect unit types for specific properties
- Missing units where required (except for 0 values)
- Using absolute units where relative units are more appropriate
- Not considering cross-browser compatibility
- Forgetting fallback values for newer formats

{dataType.category === 'color' && `
- Not providing fallbacks for modern color spaces
- Using insufficient color contrast
- Relying solely on color to convey information
`}

{dataType.category === 'length' && `
- Mixing px and em/rem inconsistently
- Using px for font-size (limits user zoom)
- Not accounting for different viewport sizes
`}

- Some value types are more performant than others

- Use CSS custom properties for values used multiple times
- Consider browser rendering optimizations

- Don't rely solely on color to convey information
